# Kiosk-NFC-Demo
This demo is to show how the following plugins work together
- Cordova Kiosk Mode - https://github.com/hkalina/cordova-plugin-kiosk
- PhoneGap NFC Plugin - https://github.com/chariotsolutions/phonegap-nfc

To install you just need to run npm install. After you have the app running on your device you need to make sure that the app is the default launcher. For more information on how to make the app the default launcher you can follow the guide on Cordova Kiosk Mode - https://github.com/hkalina/cordova-plugin-kiosk
